# Theoretical Knowledge
1-What is the main difference between "Computer Vision" and classical "Image Processing"?
- The main difference is whether the systems understand the image or not. While the input is an image in both; the output in image processing is another image, whereas in computer vision, the output is the meaning, data, or decision extracted from that image.

2-Why is it almost always necessary to convert an image to Grayscale (black and white) before applying edge detection algorithms?
- Edge detection algorithms focus on sudden changes in pixel brightness and intensity to find object boundaries, rather than color tones. Converting an image to Grayscale eliminates misleading color noise and reduces the data size to $1/3$, allowing the algorithms to process the image much faster.

3-What does an OCR (Optical Character Recognition) engine do?
- An OCR engine detects text areas within an image or scanned document and distinguishes letters and words from their pixel representations. It then converts these visual characters into a string format that can be edited, searched, and copied by a computer.
